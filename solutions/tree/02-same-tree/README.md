# Same Tree

- **LeetCode:** [100. Same Tree](https://leetcode.com/problems/same-tree/)
- **NeetCode 練習:** [Blind 75](https://neetcode.io/problems?list=blind75)

## 題意

判斷兩棵二元樹是否結構與數值完全相同。

## 思路

- **遞迴：** 兩者皆空則 True；一空一非空或值不同則 False；否則遞迴比較左子樹與右子樹。

## 時間 / 空間複雜度

- **時間:** O(n)。
- **空間:** O(h)。

## 相關閱讀

- **資料結構:** Binary Tree、遞迴

# Invert Binary Tree

- **LeetCode:** [226. Invert Binary Tree](https://leetcode.com/problems/invert-binary-tree/)
- **NeetCode 練習:** [Blind 75](https://neetcode.io/problems?list=blind75)

## 題意

將二元樹左右子樹對調（鏡像）。

## 思路

- **遞迴：** 交換根的左右子樹指標，再分別對左、右子樹做反轉。

## 時間 / 空間複雜度

- **時間:** O(n)。
- **空間:** O(h)。

## 相關閱讀

- **資料結構:** Binary Tree、遞迴

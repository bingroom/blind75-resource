# System Design — Hello Interview 向

本區為 **Nice to have**：與 Blind 75 題目對應的「系統設計 / 面試向」補充，方便在面試中從「演算法題」延伸到「系統怎麼做」。

## 可對應的主題（範例）

| 演算法 / 題型 | 可延伸的系統設計話題 |
|---------------|----------------------|
| **Hash / Two Sum** | 分散式快取、一致性雜湊、Session 儲存 |
| **LRU / Cache** | 快取策略、CDN、Redis 淘汰策略 |
| **Trie / 字串搜尋** | 搜尋建議、自動完成、日誌檢索 |
| **Heap / Top K** | 熱門排行、推薦系統、監控告警 Top N |
| **BFS/DFS、Graph** | 社交關係、推薦圖、權限/依賴拓撲 |
| **設計資料結構**（Trie, Median Finder） | API 設計、資料模型、擴展性 |

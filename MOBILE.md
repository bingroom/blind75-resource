# 手機閱讀說明

本資源已盡量配合**手機閱讀**排版，使用方式如下。

## 結構

- **一題一資料夾**：每題有 `solution.py`（可複製提交）與 `README.md`（思路、複雜度、相關閱讀）。
- **主目錄**：[README.md](README.md) 依類別列出所有題目，點連結即可進入該題。
- **manifest**：[manifest.json](manifest.json) 為機器可讀的題單，含 LeetCode 連結。

## 建議

1. **用 NeetCode 練題**：到 [NeetCode Blind 75](https://neetcode.io/problems?list=blind75) 選題，寫完後可把對應的 `solution.py` 整段複製貼上提交，確認 AC。
2. **複習**：在手機上打開該題的 `README.md` 看思路與相關閱讀即可，無需一直開電腦。
3. **字體**：若在 GitHub / 編輯器用手機瀏覽，可放大字體或使用「閱讀模式」以減少橫向捲動。

## 複雜度標註

每題 README 與程式註解中都有**時間複雜度**與**空間複雜度**，方便快速複習。
